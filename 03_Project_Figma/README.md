# Brand Filters - 产品筛选页面

这是一个基于 Figma 设计概念还原的响应式产品筛选页面。项目使用原生 HTML5, CSS3 和 JavaScript 构建，没有使用第三方框架，确保代码轻量且易于理解。

## 项目结构

```
/
├── index.html          # 主页面结构
├── css/
│   └── style.css       # 样式文件 (包含响应式设计)
├── js/
│   └── main.js         # 交互逻辑 (筛选、排序、Mock数据)
├── images/
│   └── logo.svg        # Logo 文件
└── README.md           # 项目说明
```

## 功能特性

1.  **响应式布局**：
    *   适配桌面端 (Desktop)、平板 (Tablet) 和移动端 (Mobile)。
    *   移动端侧边栏筛选器自动转换为抽屉式 (Drawer) 菜单。
    *   产品网格根据屏幕宽度自适应列数 (1-4列)。

2.  **产品筛选与排序**：
    *   **品牌筛选**：支持多选品牌 (Nike, Adidas, Puma, Reebok)。
    *   **价格筛选**：实时价格滑块筛选。
    *   **排序功能**：支持按价格 (高/低) 和最新上架排序。
    *   **动态计数**：实时显示当前筛选结果的数量。

3.  **交互体验**：
    *   平滑的过渡动画。
    *   图片懒加载 (Lazy loading)。
    *   移动端侧边栏手势/点击遮罩关闭。
    *   空状态提示 (无匹配结果时)。

## 如何运行

本项目为静态网页，您可以直接在浏览器中打开 `index.html` 文件即可预览。

或者使用本地服务器运行 (推荐，以避免跨域问题)：

```bash
# 使用 Python
python3 -m http.server

# 或者使用 npm serve
npx serve .
```

## 设计说明

由于无法直接访问 Figma 源文件 API，本项目采用了通用的现代化电商筛选页面设计规范：
*   **字体**：使用 Inter 字体，清晰易读。
*   **配色**：黑白灰为主色调，突出产品图片。
*   **图片**：使用 Placehold.co 提供的占位图服务，模拟真实产品展示。

## 浏览器兼容性

支持所有现代浏览器 (Chrome, Firefox, Safari, Edge)。
